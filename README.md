Rave By ID is a sleek, modern music streaming and library management web application designed for seamless audio playback and personalized user experiences. Built with a high-fidelity dark-themed interface, the platform allows users to browse curated feeds, track their listening insights, and manage their music collections through custom uploads and playlist organization.

Features
Persistent Media Player: A fully-integrated bottom playback bar featuring real-time progress tracking, play/pause toggles, track skipping, and precise volume control.

Personalized Analytics Dashboard: A "For You" section equipped with dynamic cards tracking listening statistics, such as total tracks and active listening minutes.

Comprehensive Library Management: Organized sidebar navigation that separates discovery tools (Feed, Playlists) from dedicated personal spaces (My Library, Uploaded, Favourites).

Local Audio Uploads: A seamless asset management flow allowing users to upload personal music files directly into their digital library.

Instant Filtering & Search: A global search bar for tracks and artists, paired with quick-toggle tab filters (All Tracks, Liked) for effortless catalog navigation.

 Stack
Frontend: javascript

Styling: Tailwind CSS

State Management / Audio Engine: HTML5 Audio API / Context API

Icons: tailwind / heroicon Icons
